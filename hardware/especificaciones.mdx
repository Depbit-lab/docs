## Tabla de especificaciones

| Parámetro | Detalle |
| --- | --- |
| Procesador seguro | Microcontrolador ARM Cortex-M33 con TrustZone |
| Criptografía | AES-256, SHA-512, ECC P-256, TRNG certificado |

| Certificaciones | FIDO2 L2, CC EAL5+, NIST SP800-63-3 |

## Indicadores LED

<Alert title="Interpretación rápida">
  El comportamiento del LED ayuda a diagnosticar el estado del dispositivo sin necesidad de software adicional.
</Alert>

| Estado | Patrón | Acción recomendada |
| --- | --- | --- |
| Inicialización | Verde intermitente | Espera la notificación de dispositivo listo. |
| Autenticación solicitada | Azul fijo | Solicita interacción del usuario (PIN o toque capacitivo). |
| Error de PIN | Rojo intermitente rápido | Verifica el PIN y los intentos restantes. |
| Firmware en actualización | Amarillo pulsante | No desconectes la llave hasta que el LED apague. |

## Seguridad física

- Carcasa resistente a manipulaciones con detección de apertura.
- Sensor de temperatura que bloquea el dispositivo ante variaciones bruscas.
- Autodestrucción de claves maestras tras 10 intentos fallidos consecutivos.
- Soporte para cadena de custodia mediante etiquetas NFC opcionales.
