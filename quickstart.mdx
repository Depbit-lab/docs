## Flujo recomendado

Sigue estos pasos para preparar un equipo piloto con ZeroKeyUSB y validar el flujo de autenticación multifactor.

<StepGroup>
  <Step title="Verifica requisitos">
    - Windows 10/11, macOS 13+ o Linux Kernel 5.15 con puertos USB-A o USB-C disponibles.
    - Privilegios de administrador para instalar controladores y el agente ZeroKey.
    - Acceso a la consola de identidad (Azure AD, Okta, Google Workspace u otro IdP compatible con FIDO2).
  </Step>
  <Step title="Instala el agente">
    1. Descarga el instalador correspondiente desde el portal de clientes.
    2. Ejecuta el instalador y acepta los permisos requeridos.
    3. Finalizada la instalación, confirma que el servicio **ZeroKey Agent** está en ejecución.
  </Step>
  <Step title="Registra el dispositivo">
    1. Conecta la llave ZeroKeyUSB al equipo.
    2. Abre la aplicación ZeroKey y selecciona **Agregar nueva llave**.
    3. Asigna un identificador al dispositivo y vincúlalo al usuario final.
    4. Define una política inicial (PIN obligatorio, intentos máximos, expiración de sesión).
  </Step>
  <Step title="Prueba el acceso">
    1. Cierra sesión en el equipo o aplicación objetivo.
    2. Inicia sesión de nuevo seleccionando **ZeroKeyUSB** como método de autenticación.
    3. Inserta la llave, ingresa el PIN y verifica que el acceso se concede correctamente.
  </Step>
</StepGroup>
