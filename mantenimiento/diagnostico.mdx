## Panel de salud

La consola ofrece un panel con métricas en tiempo real:

- Porcentaje de dispositivos en línea.
- Versiones de firmware distribuidas.
- Intentos fallidos de autenticación por usuario.
- Alertas críticas (firmware no autorizado, intentos de manipulación).

<Alert title="Objetivo recomendado">
  Mantén más del 95% de dispositivos sincronizados en las últimas 24 horas para garantizar cobertura de políticas.
</Alert>

## Comandos de diagnóstico

```bash
# Listar dispositivos con errores recientes
zerokey cli device list --status=error --since=24h

# Obtener bitácora detallada
zerokey cli device logs --device=ZK-2045 --tail=50

# Ejecutar prueba de hardware
zerokey cli device test --device=ZK-2045 --suite=full
```

## Auditoría periódica

1. Exporta el inventario completo con `zerokey cli device export --format=csv`.
2. Verifica discrepancias contra tu CMDB corporativa.
3. Revisa los registros de revocación y reasignación.
4. Documenta hallazgos y plan de acción en tu comité de seguridad.
