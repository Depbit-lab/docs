## Welcome

**ZeroKeyUSB** is a standalone, hardware-based password manager designed to keep your credentials completely offline.  
It works like a USB keyboard — automatically typing your encrypted passwords, usernames, and 2FA codes wherever you need them.  
No apps. No cloud. No subscriptions.

## Main Features

<Columns cols={2}>
  <Card
    title="Offline Encryption"
    icon="lock"
    href="/firmware/security"
  >
    Uses AES-128 CBC encryption with unique IV generation per device.  
    Your credentials never leave the hardware.
  </Card>

  <Card
    title="Hardware Simplicity"
    icon="microchip"
    href="/hardware/overview"
  >
    No batteries, no internet — powered by USB-C and built around a SAMD21 microcontroller and M24C64-W EEPROM.
  </Card>

  <Card
    title="Universal Keyboard Output"
    icon="keyboard"
    href="/firmware/utils"
  >
    Emulates a standard USB keyboard with support for multiple layouts (US, ES, FR, DE...).
  </Card>

  <Card
    title="Integrated OLED Display"
    icon="display"
    href="/firmware/display"
  >
    Navigate your credentials and 2FA codes with a clear 128×32-pixel screen.
  </Card>

  <Card
    title="Two-Factor Authentication (TOTP)"
    icon="clock"
    href="/firmware/totp"
  >
    Generate 2FA codes offline using HMAC-based algorithms (SHA-1/256/512).
  </Card>

  <Card
    title="Open Source & Transparent"
    icon="code"
    href="https://github.com/Depbit-lab/zerokeyusb"
  >
    Every line of code and schematic is public.  
    Verify what runs on your device — and contribute to improve it.
  </Card>
</Columns>
