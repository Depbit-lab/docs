# Bienvenido a ZeroKeyUSB

ZeroKeyUSB es una llave de seguridad USB diseñada para proteger el acceso a tus aplicaciones críticas mediante autenticación multifactor y almacenamiento cifrado de credenciales. En esta documentación encontrarás todo lo necesario para instalar, configurar y administrar el dispositivo dentro de tu organización.

<Alert title="Información clave">
  Esta guía está dirigida a equipos de TI, administradores de seguridad y desarrolladores que integran ZeroKeyUSB con sus plataformas.
</Alert>

## Capacidades destacadas

- **Autenticación resistente al phishing** mediante tokens FIDO2 y OTP.
- **Almacenamiento seguro** con cifrado AES-256 y protección contra manipulación.
- **Gestión centralizada** para distribuir políticas, firmware y auditorías.
- **Integraciones listas** con sistemas operativos modernos y proveedores de identidad.
