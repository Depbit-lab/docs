---
title: "Referencia de comandos"
description: "Interfaz CLI y API serie del ZeroKeyUSB"
---

La suite ZeroKey proporciona una interfaz de línea de comandos (CLI) y un canal serie seguro para automatizar tareas de administración. Todos los comandos requieren autenticación con un token administrativo generado en la consola.

## Autenticación

```bash
# Establece el token para la sesión actual
export ZEROKEY_TOKEN="<token_api>"
```

Cada comando devuelve códigos de salida normalizados: `0` para éxito, `1` para error de validación y `2` para fallas de comunicación.

## Formato de respuesta

Las operaciones responden en JSON. Ejemplo:

```json
{
  "device": "ZK-2045",
  "status": "online",
  "mode": "fido2",
  "lastSync": "2024-03-21T10:15:00Z"
}
```

Continúa con los comandos disponibles para gestionar dispositivos, firmware y auditoría.
