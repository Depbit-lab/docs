---
title: "Obtener estado del dispositivo"
description: "Consulta la telemetría actual de un ZeroKeyUSB"
---

```bash
zerokey cli device status --device=<serie>
```

## Parámetros

| Nombre | Tipo | Obligatorio | Descripción |
| --- | --- | --- | --- |
| `device` | string | Sí | Número de serie del ZeroKeyUSB (formato ZK-XXXX). |
| `output` | enum(json, table) | No | Formato de respuesta. Por defecto `json`. |

## Respuesta exitosa

```json
{
  "device": "ZK-2045",
  "status": "online",
  "firmware": "2.3.1",
  "mode": "fido2",
  "lastSeen": "2024-03-21T10:15:00Z"
}
```

## Errores comunes

| Código | Mensaje | Acción |
| --- | --- | --- |
| 404 | Device not found | Verifica el número de serie o sincroniza el inventario. |
| 423 | Device locked | Revisa las políticas de bloqueo y el historial de intentos fallidos. |
| 500 | Agent unreachable | Comprueba la conectividad entre el agente y la consola.
